# Grammar Scoring Engine

## Project Overview
The Grammar Scoring Engine is designed to evaluate and score grammar usage in written texts. This project aims to assist users in improving their writing through accurate feedback on grammatical errors and suggestions for corrections.

## Architecture
The architecture of the Grammar Scoring Engine consists of several modules that interact to provide grammar analysis. The main components include:
- **Input Module**: Receives the text content from the user.
- **Processing Module**: Analyzes the text for grammatical errors using natural language processing techniques.
- **Output Module**: Sends feedback and scores back to the user, highlighting errors and suggestions.

## Setup Instructions
To set up the Grammar Scoring Engine, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/GreenCodr/shl-internship.git
   ```
2. Navigate to the project directory:
   ```bash
   cd shl-internship
   ```
3. Install necessary dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the application:
   ```bash
   python main.py
   ```

## Usage Guide
To use the Grammar Scoring Engine:
1. Input your text into the provided text area in the application.
2. Click the "Score" button to evaluate the text.
3. Review the feedback provided, including error highlights and score.
4. Make corrections as suggested and resubmit for further evaluation.
